# Shift Manager App

A full-stack web application built with Next.js and Firebase for managing work shifts, tracking hours, and calculating salary — designed specifically for the Israeli labor market.

---

## Live Demo

[View Live App](https://hamza-project-five.vercel.app/)

---

##  Features

- ✅ User authentication (Sign In / Sign Up)
- ✅ Add, edit, and delete work shifts
- ✅ Morning, Evening, Night, and Custom shift types
- ✅ Automatic salary calculation based on hourly rate
- ✅ Night shift rate multiplier (125% / 150%) — Israeli labor law
- ✅ Weekly and monthly earnings overview
- ✅ Interactive earnings chart (Area Chart)
- ✅ Shift breakdown by type (Pie Chart)
- ✅ Built-in calendar with shift indicators
- ✅ Upcoming shifts view
- ✅ Break duration deduction
- ✅ Settings page with customizable hourly rate
- ✅ Fully responsive — works on mobile and desktop

# Environment Variables

Create a `.env.local` file in the root directory:

```env
NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=
NEXT_PUBLIC_FIREBASE_DATABASE_URL=
```

---

# Getting Started

```bash
# Install dependencies
npm install
npm firebase install
# Run development server
npm run dev

# Build for production
npm run build
```
